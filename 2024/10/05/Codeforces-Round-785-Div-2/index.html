Jeep Yu
2024-10-05

Problem - A - Codeforces

题意：

给你一个字符串，$alice$可以删除偶数个连续的字符，$bob$可以删除奇数个连续的字符，$alice $先删除。每删除一个字符$cost+=s[i]-‘a’+1$,问最后谁赢，两者差值是多少。

思路：

假如字符串长度为偶数，那一定是alice赢。答案就是整个字符串的$cost$。假如是奇数，alice把它删除的只剩下一个，这个字符在s[0]和s[n-1]中选。让bob选择小的那个。最后得出两者的值，比较得出答案。

Problem - B - Codeforces

题意：

给你一个字符串，要求对于这个字符串里面所有的子串，都要满足在这个字符串所有出现过的字符在这个子串出现过的次数相差不超过1.满足要求输出YES，否则NO。

例子：

对于字符串$abc$有：

t ft(a) ft(b) ft(c)
a 1 0 0
ab 1 1 0
abc 1 1 1
b 0 1 0
bc 0 1 1
c 0 0 1

因此$abc$是满足条件的。所有的相差值不超过1.

思路：

最简单的：对于一个字符串所有元素的都只出现了一次的，肯定是满足条件的。而出现两次的话，我们就应该对两个相同字符之间的字符进行判断，看看是不是字符串上除了两边的字符所有的字符都出现了且只出现了一次。这里我采用了对字符串里面不同的字符进行赋值并求和得出sum。因为我只需要判断里面包含所有的字符就行，不关心它的位置，因此在判断两相同字符里面时我只需求和，看看和$sum-(s[i]-‘a’+1)$是否相同。不同就说明不满足条件。对于找前一个位置相同的字符，我用一个栈来存它，每次都$push$进去，要比对的时候取$top$出来。

Problem - C - Codeforces

题意：

求一个数被回文数组成的方法数。$(n\leq 4\times 10^4)$

例子：对于$n=5$:

  • $5=1+1+1+1+1$
  • $5=1+1+1+2$
  • $5=1+2+2$
  • $5=1+1+3$
  • $5=2+3$
  • $5=1+4$
  • $5=5$

思路：

这道题我当时一看就觉得像是dp，可惜这部分的dp题做的不够多，所以是dp也不会。

这个n不大，1e4，其实是可以大遍历预处理然后得出答案的。怎么设置dp捏？而且我们可以(然而我看不出来)看出回文数并不多，就在1e4以下的也就499个，因此我们先遍历一遍把回文数数组得出来，然后我们让i表示为我们要求的数，j表示为对于回文数的前j个，dp值就是次数。那怎么写状态方程捏？要么选要么不选嘛。选不了的时候，$dp[i][j]=dp[i][j-1]$，选的了的时候：$dp[i][j]=dp[i][j-1]+dp[i-v[j]][j]$。

对于j的遍历，因为我们的$dp[i][j]=dp[i][j-1]+dp[i-v[j]][j]$,因此我们得从$j=1$开始遍历。

Problem - D - Codeforces

题意：

这里有两个等差数组$a,b$，它们的交集得到了等差数列c。现在你知道了b,c。求a可能存在的个数，假如a不存在，输出0.假如a有无限多，输出-1.

思路：

先来看看这数据的大小，这注定是不能用数组存的，必须进行分析。

首先我们先分析下什么时候得到0.就是c中有一部分不属于b。这怎么得到？

  1. c的公差不是b的公差的倍数
  2. c的公差是b的公差的倍数 但是b没有把c完全包括在内

化为表达式就这样：

那什么时候等于-1捏？

**对于c1，它的c1-dc要么就是在b里面，要么就是$\leq b1$**，只要c1-dc在b的范围外，那不就是可以无穷的变大吗？同理$c1+(n-1)\times dc+dc$，假如它坐落于b的外边就输出-1.

接下来就要计算除以上两种情况下的答案，假设a公差为d3。

先讨论特殊情况：d1==d2

例如序列b,c。(c序列为括号内的元素序列)

-6 (0) (6) (12) 18 d1==d2==6

前面明确过当一个等差数列是另一个等差数列的一部分时，那么前者公差一定是后者公差的倍数

所以d3一定是d2的因子，d3可能值：1 2 3 6 显然都满足

*所以发现如果 d1==d2 那么d3的所有可能为d2的所有因子。

具体答案如何算？

假设公差为3

a序列可能是

0 3 6 9 12

0 3 6 9 12 15

-3 0 3 6 9 12

-3 0 3 6 9 12 15 四种。

a序列所有可能可以由序列 0 3 6 9 12延伸得到

延申左边的可能情况为 可延申的数字个数(d2/d3 -1) + 1 = d2/d3

右边同理，所有情况为左右两类乘积；d2/d3 * d2/d3 。

如果d1!=d2 ,假设a公差为d3

　a，b 序列包含c序列 满足： d2%d3!=0 ,d3%d2!=0 d1!=d2

*那么d3 和 d1 就不能存在倍数关系，而且 d1 和 d3的 lcm 必须是d2

有一种特殊情况：d3==d2 时虽然不满足d3%d2!=0 实际与d1并无冲突
